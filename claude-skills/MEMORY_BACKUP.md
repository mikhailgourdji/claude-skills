# Memory Backup — claude.ai

Резервная копия персональных записей памяти.
Обновлять вручную при изменениях.

**Последнее обновление:** 2026-03-20

---

## Активные записи

### 1. Телеметрия Claude Code
```
Проверить раз в неделю: отключена ли телеметрия Claude Code 
(CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1 в ~/.zshrc, 
папка ~/.claude/telemetry/ должна быть пустой)
```

### 2. Новый проект — протокол
```
При старте нового проекта с разработкой: предложить трёхуровневую 
схему Chat+Code+Cowork, определить PROJECT-ID, предложить создать 
папку <PROJECT-ID>-cowork и настроить Cowork как мост. После 
обсуждения задачи — сгенерировать TASK.md с полем Project: и 
сказать "положи в папку cowork — он сам передаст в Code".
```

---

## Как восстановить

На новом компьютере или после сброса памяти — скопируй каждую запись
и добавь через `memory_user_edits` или скажи Claude:
"Запомни: [текст записи]"
